# Training Dashboard

A web application for managing, tracking, and reporting on training activities.

## Tech Stack

- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** SQLite
- **Hosting:** AWS Lightsail
